## Backend Integration

This project is designed to work with the moto2-api backend.

### How to Run the Backend
1. Clone the moto2-api repository.
2. Install dependencies: `npm install`.
3. Set up your .env file based on .env.example.
4. Start the server: `npm start`.

### Wiring the Frontend
To connect the frontend to the backend, ensure your environment variables are set correctly in the frontend's .env file. The frontend expects the backend to be running on the URL specified in VITE_API_URL.
